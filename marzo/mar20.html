20/03 DIAGRAMA DEL FUNCIONAMIENTO DETALLADO DEL ORDENADOR

1. Entrada de Datos

  • Usuario ingresa datos a través de dispositivos de entrada (teclado, mouse, etc.)
  • Los datos se envían al Procesador para su procesamiento

2. Procesador (CPU)

  • Recibe instrucciones y datos del Sistema Operativo y de los programas
  • Ejecuta instrucciones y realiza cálculos
  • Almacena resultados en la Memoria RAM

3. Memoria RAM

  • Almacena datos y programas temporalmente mientras el ordenador está encendido
  • Proporciona acceso rápido a los datos y programas que el Procesador necesita

4. Disco Duro (HD) o Unidad de Estado Sólido (SSD)

  • Almacena datos y programas a largo plazo
  • Proporciona almacenamiento permanente para el Sistema Operativo, programas y datos

5. Tarjeta Gráfica

  • Procesa imágenes y video para su visualización en la pantalla
  • Recibe instrucciones del Procesador y envía la imagen procesada a la pantalla

6. Salida de Datos

  • Los resultados procesados se envían a dispositivos de salida (pantalla, impresora, etc.)
  • El usuario puede ver los resultados en la pantalla o imprimirlos

7. Sistema Operativo

  • Gestiona los recursos del ordenador y proporciona una interfaz para el usuario
  • Coordina la ejecución de programas y el acceso a los dispositivos de hardware

Conexiones

  • Procesador -> Memoria RAM: El procesador accede a los datos y programas almacenados en la memoria RAM.
  • Memoria RAM -> Disco Duro (HD) o Unidad de Estado Sólido (SSD): La memoria RAM carga datos y programas desde el disco duro o SSD.
  • Procesador -> Tarjeta Gráfica: El procesador envía instrucciones a la tarjeta gráfica para procesar imágenes y video.
  • Tarjeta Gráfica -> Pantalla: La tarjeta gráfica envía la imagen procesada a la pantalla para su visualización
